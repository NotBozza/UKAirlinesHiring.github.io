# ✈️ UK Pilot Jobs — Hiring Tracker

A modern, real-time tracker for pilot hiring across UK airlines.
Built to give aspiring and current pilots clear, up-to-date insight into airline recruitment status.

---

## 🌐 Live Website

👉 https://ukpilotjobs.co.uk

---

## 📊 Features

* ✅ Track UK airline hiring status in one place
* 🔍 Filter by:

  * Open Now
  * Pathway Programs
  * Watchlist
  * Not Hiring
* 📡 “Live Intel” style interface for quick scanning
* 🎯 Clear notes on each airline’s recruitment process
* 💻 Fully responsive, modern UI

---

## 🎨 Design Philosophy

The site is designed with a **clean, aviation-tech aesthetic**, inspired by:

* Airline operations dashboards
* Glass cockpit UI styling
* Dark mode for reduced eye strain

Focus is on:

* Clarity
* Speed
* Real-world usefulness

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (custom styling, no frameworks)
* GitHub Pages (hosting)
* Custom domain via IONOS
